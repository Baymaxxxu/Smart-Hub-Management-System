# Email Notification Feature Plan

Fitur ini disiapkan untuk mengirimkan notifikasi email kepada anggota ketika status peminjaman berubah.

## Rencana Fitur

- Email ketika peminjaman disetujui
- Email ketika peminjaman ditolak
- Email pengingat sebelum jadwal peminjaman dimulai
- Email ketika equipment belum di-checkout

## Strategi Pengembangan

Fitur ini dikerjakan pada branch `feature/email-notification` agar tidak mengganggu kode utama pada branch `master`.

Setelah fitur selesai dan sudah dites, branch ini dapat digabungkan kembali ke branch `master`.
